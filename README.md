# 天原危化品车辆管理系统
基于vue/cli3.0和ant-design-vue的实现
```
测试账号 guest,guard,admin,sadmin 密码与账号相同
```
## 线上预览地址
http://39.108.95.179:8000/dist/index.html#/login

## 安装依赖
```
yarn install
```

### 项目启动
```
yarn run serve
```

### 发布
```
yarn run build
```

### 测试
```
yarn run test
```

### 代码规范
```
yarn run lint
```
